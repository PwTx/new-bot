## Clone this project 
## use the command below, if you want to download via the link

```bash
> git clone https://github.com/HasamiAini/wabot-termux-takagisan.git
```

## Install the dependencies:
follow the installation instructions below to continue,
do it carefully, to make it work

```bash
> pkg upgrade && pkg update
```
```
pkg install git

```
```
pkg install nodejs

```
```
pkg install ffmpeg -y

```
```
cd wabot-termux-takagisan

```
```
npm install gify-cli -g

```
```
npm install

```

bash install.sh

```
